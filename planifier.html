Planifier & publier

Compose un post (ou laisse l'IA le proposer), choisis les réseaux et la date : AWEMA publie tout seul le moment venu. Guide

0 caractère
AWEMA enregistre le post dans ta file d'attente (ton dépôt) et le publie au moment prévu via une GitHub Action. Image : si tu choisis un fichier, il est hébergé sur ton site et son adresse est transmise aux plateformes. Vidéo : colle une URL (YouTube/TikTok sont activés progressivement).

Génère des propositions : donne une idée, ou laisse vide pour t'appuyer sur ce qui marche le mieux sur le compte. L'agent Créatif s'en occupe.

Génération côté GitHub (~1 min) avec ta clé IA. Rien ne se passe ? Vérifie que ta clé est bien posée et que le fournisseur est choisi : Mise en route.

Propositions disponibles — clique « Utiliser » pour pré-remplir le composer.

Tes posts programmés et publiés. Le statut se met à jour tout seul après chaque passage du planificateur.
