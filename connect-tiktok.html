Connecter TikTok — guide pas à pas

Suis les étapes dans l'ordre. La seule action « manuelle » imposée par TikTok : te connecter au compte + cliquer Authorize. L'assistant fait tout le reste (capture du code, échange, mise à jour de la liste des comptes).

⏱️ ~10 min la 1ʳᵉ fois · puis 30 s par compte

1 Créer le « PAT » GitHub (clé d'automatisation)

Le PAT autorise l'assistant à enregistrer tout seul tes comptes TikTok dans le projet. C'est une clé personnelle, à créer une seule fois.

  • Ouvre la page de création (bouton ci-dessous).
  • Token name : awema-tiktok · Expiration : 90 jours (ou plus).
  • Repository access : Only select repositories → choisis AwemA---Agence-Web-Marketing-Africaine.
  • Permissions → Repository permissions → Variables → Read and write.
  • Generate token → copie le token (tu ne le reverras plus).
Créer le PAT ↗
Garde ce token privé. Ne le colle nulle part en ligne — il sert juste à la commande set GH_PAT=… de l'étape 3.

2 Autoriser l'adresse locale dans l'app TikTok

Pour que l'assistant récupère le code automatiquement, ajoute cette Redirect URI dans l'app TikTok (section URL properties / Login Kit) :

http://127.0.0.1:8723/callback
Si TikTok refuse les adresses localhost : pas grave, tu lanceras l'assistant avec l'option --paste (étape 4) — il réutilise l'adresse du projet déjà autorisée.

3 Préparer le terminal

Ouvre un terminal dans le dossier du projet et colle ces lignes (remplace TON_SECRET et TON_PAT) :

cd chemin\vers\awema---agence-web-marketing-africaine git pull set TIKTOK_CLIENT_KEY=sbaww0y3bflk3knbkj set TIKTOK_CLIENT_SECRET=TON_SECRET set GH_PAT=TON_PAT
Mac / Linux : remplace chaque set X=Y par export X=Y.

4 Lancer l'assistant

python scripts\tiktok-onboard.py

Repli si TikTok a refusé l'adresse locale (étape 2) :

python scripts\tiktok-onboard.py --paste
Si python ne marche pas, essaie py à la place.

5 Autoriser chaque compte (en boucle)

  • Dans le navigateur, connecte-toi au compte TikTok à ajouter.
  • Reviens au terminal, appuie sur Entrée → le navigateur s'ouvre → clique Authorize.
  • L'assistant affiche le nom du compte et propose un slug (nom de dossier client) → Entrée.
  • « Ajouter un autre ? » → change de compte TikTok dans le navigateur et recommence.
  • Quand c'est fini, tape q → l'assistant met à jour la liste des comptes tout seul.
Important : c'est le compte connecté dans le navigateur au moment du clic qui est capturé. Change donc de compte entre chaque autorisation.

6 Lancer la synchronisation

Va sur la page du workflow et clique Run workflow → tous les comptes se synchronisent, et leurs vraies stats apparaissent dans le dashboard.

Ensuite, c'est automatique chaque lundi : le PAT permet de rafraîchir et resauvegarder les jetons sans aucune intervention.
