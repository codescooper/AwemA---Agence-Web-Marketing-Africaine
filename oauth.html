AwemA Watcher · code d'autorisation TikTok

Cette page récupère le code renvoyé par TikTok après autorisation. Copie-le, puis échange-le en ligne de commande.

Étape suivante (à coller dans ton terminal) :
export TIKTOK_CLIENT_KEY="…" TIKTOK_CLIENT_SECRET="…"
python3 scripts/connect-reseaux.py --tiktok-auth "<CODE>" "<REDIRECT_URI>"
→ tu obtiens le refresh_token à mettre dans la Variable TIKTOK_TOKENS.
