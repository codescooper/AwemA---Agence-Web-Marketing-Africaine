Connexion TikTok · finalisation

TikTok t'a renvoyé ici après l'autorisation. AWEMA termine tout seul : il déclenche l'échange sécurisé du code en arrière-plan (GitHub Actions). Aucun terminal.

Plutôt en ligne de commande (si tu as Python en local)
export TIKTOK_CLIENT_KEY="…" TIKTOK_CLIENT_SECRET="…"
python3 scripts/connect-reseaux.py --tiktok-auth "<CODE>" "<REDIRECT_URI>"
→ tu obtiens le refresh_token à mettre dans la Variable TIKTOK_TOKENS. Le redirect_uri à utiliser est l'adresse de cette page.
