Mise en route

Configure tes APIs et ressources pour faire tourner AWEMA à 100 %. Tu ne connectes que les réseaux de tes clients. L'app suit ta progression et te dit quoi faire.

0 %

1 · Socle — GitHub à connecter

GitHub héberge ton instance et reçoit tes données quand tu cliques « Enregistrer ». Active aussi Settings → Pages (branche main, dossier /).

2 · Cerveau — une IA recommandé · gratuit

Branche une IA (gratuite : Groq, Gemini…) pour que les agents Analyste / Stratège / Créatif travaillent. Colle ta clé ici : AWEMA enregistre le fournisseur tout seul et te guide pour le dernier collage. Sans IA, « 3 choses à faire aujourd'hui » marche quand même.
🔑 Obtenir une clé → Comparer tous les fournisseurs →

3 · Les réseaux de tes clients

Pour chaque réseau que tes clients utilisent : crée l'App, récupère le token, mets-le en Secret GitHub (le guide te montre où). L'état ci-dessous se met à jour dès qu'un client est connecté.
⚠️ Gérer de vrais clients (pas tes comptes perso) demande, côté Meta/TikTok/LinkedIn, une App vérifiée + App Review — c'est une exigence des plateformes (incompressible). Détails & alternative « via AWEMA » : guide accès API.

4 · Tes clients & leur ADN

Ajoute tes clients, puis remplis leur Mémoire Marketing (ton, personas, produits) — c'est ce qui rend les propositions de l'IA justes.
➕ Ajouter un client 🧠 Mémoire Marketing

5 · Lancer & automatiser une fois

Dans l'onglet Actions de ton dépôt GitHub, clique Run workflow sur « Sync présence digitale » et « Sync TikTok » pour la 1ʳᵉ récupération de données. Ensuite, tout est autonome : synchro périodique → agents (quotidien + après chaque sync) → cockpit à jour.
