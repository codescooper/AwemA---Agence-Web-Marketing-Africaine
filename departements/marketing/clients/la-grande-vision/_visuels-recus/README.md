# _visuels-recus — Images générées (reçues)

Dépôt des **visuels produits** à partir des prompts (ChatGPT, Claude Desktop ou API OpenAI),
nommés par identifiant de contenu : `001.png`, `002.png`, …

- Convention : `NNN.png` (NNN = ID du contenu) ou `NNN-<plateforme>.png`.
- Pour charger un visuel dans l'outil de revue : le **glisser** sur l'aperçu, le **coller**
  (Ctrl/Cmd+V), ou indiquer son chemin dans le champ *Image du visuel*.

Voir la méthode complète : [`../../../../../outils/revue-visuels/INTEGRATION-GENERATION-IMAGE.md`](../../../../../outils/revue-visuels/INTEGRATION-GENERATION-IMAGE.md).

> Les images (`*.png`, `*.jpg`) ne sont pas versionnées (voir `.gitignore`) : ce sont des
> artefacts reproductibles à partir des prompts.
