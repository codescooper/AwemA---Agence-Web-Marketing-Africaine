# Visuels Canva produits — La Grande Vision

Designs générés dans Canva à partir des prompts (charte respectée) et **finalisés dans le
compte Canva du client**. Les liens ci-dessous sont durables (vue + édition).

> ℹ️ Les PNG ne sont pas stockés dans le dépôt : l'environnement de génération bloque le
> domaine de téléchargement Canva (`export-download.canva.com`). Pour récupérer les fichiers,
> ouvrir le design et **Partager → Télécharger (PNG)**, puis déposer les images ici si besoin.
> L'outil de revue (`outils/revue-visuels`) affiche déjà les vignettes en ligne.
