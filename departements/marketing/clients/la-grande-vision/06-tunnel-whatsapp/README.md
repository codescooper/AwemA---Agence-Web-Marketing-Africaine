# 06 — Tunnel WhatsApp (7 étapes)

Tunnel complet de conversion : de la publicité jusqu'au suivi après visite. Tous les
messages sont **rédigés et prêts à l'emploi**. Variables entre `{{ }}`.

```
Pub → Message auto → Qualification → Prise de RDV → Confirmation → Relance → Suivi après visite
```

> Canal : **WhatsApp Business**. Configurer : message d'accueil, réponses rapides
> (raccourcis), étiquettes (labels), catalogue. Les messages respectent le ton premium,
> médical et chaleureux de la charte.

---

## Étape 1 — Publicité (entrée du tunnel)

**Objectif :** générer le 1ᵉʳ message WhatsApp.
**Format :** pub Meta (Facebook/Instagram) bouton « Envoyer un message » → WhatsApp, ou
post avec mot-clé.

**Accroche pub :**
> 👓 Votre vue mérite mieux qu'une devinette.
> Bilan de vue à La Grande Vision (Yopougon). Écrivez-nous **« VUE »** sur WhatsApp,
> on s'occupe de tout. 📲

**Mot-clé d'entrée :** `VUE` ou `RDV`.

---

## Étape 2 — Message automatique (accueil)

Déclenché dès le 1ᵉʳ message.

```
Bonjour 👋 et bienvenue à La Grande Vision, votre cabinet d'optique à Yopougon ! 👓
Merci de nous avoir contactés.

Je suis l'assistant de prise de rendez-vous. En 2 minutes, on trouve la solution
adaptée à vos yeux. 😊

Pour commencer, dites-nous : c'est pour…
1️⃣ Un bilan / consultation visuelle
2️⃣ De nouvelles lunettes (montures)
3️⃣ Des verres progressifs
4️⃣ Autre chose

Répondez simplement par 1, 2, 3 ou 4.
```

---

## Étape 3 — Qualification

But : comprendre le besoin et le persona.

```
Parfait, merci ! 🙏 Encore quelques questions rapides :

A) Pour qui est-ce ? (vous / un enfant / un proche)
B) Avez-vous déjà des lunettes ? (oui / non)
C) Quel est votre principal souci ? (vision de loin / de près / écrans / maux de tête / autre)
D) Vous êtes plutôt disponible : en semaine ou le week-end ?

Vos réponses nous aident à préparer votre visite. 👌
```

**Réponse selon le besoin (exemples de personnalisation) :**

- *Écrans / maux de tête (Koffi)* :
```
Je comprends 👀 La fatigue des écrans est très fréquente. Un bilan permet de voir si des
verres anti-lumière bleue vous soulageraient. On vous propose un rendez-vous rapide ?
```
- *Vision de près après 40 ans (Mariam)* :
```
C'est souvent la presbytie, très courante et bien corrigée par des verres progressifs
confortables. On en parle lors d'un bilan ? 😊
```
- *Enfant (Awa)* :
```
Très bonne démarche 🙌 La vue des enfants influence leur réussite scolaire. Le bilan enfant
est simple et rapide. On vous trouve un créneau ?
```

(Et si vous avez une ancienne ordonnance, apportez-la 📄)

## Étape 6 — Relance (rappels & no-show)

**Rappel J-1 :**
```
Bonjour {{PRENOM}} 👋 Petit rappel : votre bilan à La Grande Vision, c'est demain
{{DATE}} à {{HEURE}} 🕐. On a hâte de vous accueillir !
Si besoin de décaler, dites-le nous, c'est sans souci. 😊
```

**Rappel jour J (matin) :**
```
Bonjour {{PRENOM}} ☀️ C'est le jour J ! RDV à {{HEURE}} à Yopougon.
📍 Itinéraire : {{LIEN_MAPS}}. À tout à l'heure ! 👓
```

**Relance si pas venu (no-show, +2h) :**
```
Bonjour {{PRENOM}}, nous ne vous avons pas vu aujourd'hui 😊 Pas de souci, ça arrive !
Souhaitez-vous reprogrammer votre bilan ? Voici d'autres créneaux : {{CRENEAUX}}.
Votre vue compte pour nous. 🙏
```

**Relance prospect non converti (+3 jours) :**
```
Bonjour {{PRENOM}} 👋 Vous nous aviez écrit pour un bilan de vue. L'offre tient toujours !
Voulez-vous qu'on vous réserve un créneau cette semaine ? Répondez « OUI ». 😊
```

## Étape 7 — Suivi après visite

**J+1 (satisfaction) :**
```
Bonjour {{PRENOM}} 😊 Merci pour votre visite à La Grande Vision !
Comment se passent vos premiers jours ? Tout est confortable ?
(N'hésitez pas si le moindre réglage est nécessaire, c'est gratuit 👍)
```

**J+2 (avis Google) :**
```
Si votre expérience vous a plu, votre avis nous aiderait énormément ⭐⭐⭐⭐⭐
👉 {{LIEN_AVIS_GOOGLE}}
Merci infiniment, ça compte beaucoup pour un cabinet de quartier ! 🙏
```

**J+7 (confort & vente complémentaire) :**
```
Bonjour {{PRENOM}} 👋 Une semaine avec vos nouvelles lunettes ! Tout va bien ?
💡 Astuce : une 2ᵉ paire (soleil à votre vue / lecture) change le quotidien.
Dites-nous si ça vous intéresse, on a des offres. 😊
```

**Fidélisation (rappel annuel) :** voir [`../07-crm-relance/`](../07-crm-relance/).
