# 03 — Contenus rédigés (180)

[`contenus.md`](contenus.md) contient les **180 fiches de contenu**, générées par
`_generateur/generer.py`. Correspond au **Google Doc** des livrables.

Chaque fiche contient : Titre, Objectif, Persona, Pilier, Format & Visuel recommandés,
KPI attendu, Hook, **Texte Facebook**, **Texte Instagram**, **Texte LinkedIn**,
**Description TikTok**, CTA, Hashtags, **Prompt Canva**, **Prompt Midjourney**,
**Prompt GPT Image**.

> ⚠️ Ne pas éditer `contenus.md` à la main : il est régénéré. Pour modifier le fond,
> éditer `_generateur/sujets.py` (banque de sujets) ou `_generateur/generer.py` (gabarits),
> puis relancer la génération.

## Régénérer
```bash
cd ../_generateur && python3 generer.py
```
