## Règle absolue — Charte

Tout prompt doit inclure le **bloc charte** (couleurs + typo + mood). Bloc prêt à coller :

```
Palette: deep navy blue #0A1F44 (dominant), sky blue #4BA3FF (accent), gold #D4AF37 (premium).
Typography: Montserrat (headlines) + Poppins (body), clean modern medical.
Mood: premium, modern, medical, trustworthy, family-friendly, professional.
Lighting: bright soft natural. Composition: clean, airy, generous white space.
Context: optical/eyewear clinic in Yopougon, Abidjan, Côte d'Ivoire.
```

## Structure imposée des prompts

- **Canva :** Format · Palette · Typographie · Composition · Visuels · CTA.
- **Midjourney :** Sujet · Style · Éclairage · Caméra · Objectif · Composition · Couleurs ·
  Détail — suffixe `--ar 4:5 --stylize 250`.
- **GPT Image :** version réaliste, photographie professionnelle, compatible publicité Meta.
